# Weather Web Application

A basic weather web app that shows the current temperature and a 5 day forcast for any city in the world.

## Features
- Cuurent weather data using OpenWeatherMap API
- 5 day forecast display
- Humidity, wind speed, and feels-like temperature
- Mobile friendly design

## Tech Stack
JavaScript, HTML, CSS, REST API (OpenWeatherMap)

## Setup
- Clone the repo
- Get a free API key at openweathermap.org
- Put API key in `app.js`
- Open `index.html` in a browser
